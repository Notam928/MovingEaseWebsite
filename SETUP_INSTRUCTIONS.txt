QUICK START GUIDE
================================================================

Step 1: INSTALL NODE.JS (if you don't have it)
-----------------------------------------------
1. Go to: https://nodejs.org/
2. Download the LTS version (Long Term Support)
3. Run the installer
4. Open Terminal/Command Prompt and verify:
   
   node --version
   npm --version

   You should see version numbers appear.


Step 2: EXTRACT THE ZIP FILE
-----------------------------
1. Extract "movers-website-project.zip" to your desired location
2. Remember the folder location


Step 3: OPEN TERMINAL IN PROJECT FOLDER
----------------------------------------
Windows:
  - Open the folder in File Explorer
  - Type "cmd" in the address bar
  - Press Enter

Mac/Linux:
  - Open Terminal
  - Use 'cd' command to navigate:
    cd /path/to/movers-website-project


Step 4: INSTALL DEPENDENCIES
-----------------------------
In the terminal, run:

npm install

This will take 2-5 minutes. Wait until it completes.


Step 5: START THE WEBSITE
--------------------------
After installation completes, run:

npm start

The website will automatically open in your browser at:
http://localhost:3000

If it doesn't open automatically, manually open your browser
and go to: http://localhost:3000


================================================================
TROUBLESHOOTING
================================================================

Problem: "npm: command not found"
Solution: Node.js is not installed. Go back to Step 1.

Problem: "Port 3000 is already in use"
Solution: Either:
  - Close the other app using port 3000, OR
  - The terminal will ask if you want to use another port. Type 'y' and press Enter.

Problem: Installation fails
Solution: Try:
  npm cache clean --force
  npm install

Problem: "Module not found" errors
Solution: Make sure you ran "npm install" first.

Problem: Website doesn't look right
Solution: 
  - Clear your browser cache (Ctrl+Shift+Delete)
  - Refresh the page (Ctrl+R or Cmd+R)

================================================================
CUSTOMIZATION
================================================================

To change content:
1. Open: src/App.jsx
2. Search for the text you want to change
3. Save the file
4. The browser will automatically refresh

To change colors:
1. Open: src/App.jsx
2. Find color classes like: "bg-blue-600" or "text-orange-500"
3. Replace with your preferred Tailwind colors
4. Save and the browser will refresh

To add your own images:
1. Replace the Unsplash image URLs with your own
2. Format: <img src="your-image-url.jpg" alt="description" />
